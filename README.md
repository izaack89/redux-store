## Installation

Use the package manager [npm](https://docs.npmjs.com/cli/v7/commands/npm-install) to install all the dependencies that are on the package.json

```bash
npm install
```

## Usage

To Run the system locally you need to use the follow command

```bash
npm run develop
```
